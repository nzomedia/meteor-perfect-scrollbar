# Perfect Scrollbar

https://noraesae.github.io/perfect-scrollbar/ version 0.6.16 packaged for **Meteor**


```
meteor add nzomedia:perfect-scrollbar
```
___


## Example Usage
#### JavaScript:
```js
$(function() {
    $('#Demo').perfectScrollbar();
});
```

#### HTML:
```html
<div id="Demo">
  <div class="item">......</div>
  <div class="item">......</div>
  <div class="item">......</div>
</div>
```
#### Css:
```sass
#demo {
  position: relative; // Required
  height: 200px; // Optional
}
```
